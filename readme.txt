Assignment 2 Grade Calculator:

Features:
1. Set grades for homework assignments and exams
using horizontal scroll bars and spin boxes.

2. Switch between calculators for different courses
(between PIC 10B and PIC 10C) using checkboxes.

3. Select between two grading schemes via checkboxes
for each of the two courses.

4. Get information on the grading schemes using a
push button that is marked with a "?"

5. Have LCD Number displays for the homework score,
the exam score, and the overall score.

6. Has a calculate button for when the user finishes
inputting their information.

7. Has a reset button to return to the start state
of the program.

8. Uses some horizontal and vertical lines to separate
different sections of the calculator.
